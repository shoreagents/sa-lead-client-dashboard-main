=====================================
INSTALLATION STEPS
=====================================

Option A: Automated (Recommended)
----------------------------------
1. Extract this package to a folder
2. Open terminal in that folder
3. Run: npm install
4. Run: npm run setup
5. Run: npm run echo-claude

Option B: Manual
----------------------------------
1. Extract this package to a folder
2. Open terminal in that folder
3. Run: npm install
4. Copy env.template to .env
5. Edit .env with your API keys
6. Run: npm run echo-claude

=====================================
QUICK COMMANDS
=====================================

npm install              Install dependencies
npm run setup           Run setup wizard
npm run echo-claude     Start Echo bot

pm2 start echo-claude-bot.js --name echo   Run 24/7
pm2 logs echo                              View logs
pm2 restart echo                           Restart
pm2 stop echo                              Stop
