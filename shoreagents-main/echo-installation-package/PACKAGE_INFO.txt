=====================================
WHAT YOU NEED
=====================================

Before installation, obtain:

1. Slack Bot Token (xoxb-...)
   From: https://api.slack.com/apps
   Required Scopes:
   - channels:history
   - channels:read
   - chat:write
   - users:read

2. Claude API Key (sk-ant-api03-...)
   From: https://console.anthropic.com/
   Any active key works

=====================================
INSTALLATION STEPS
=====================================

Option A: Automated (Recommended)
----------------------------------
1. Extract this package to a folder
2. Open terminal in that folder
3. Run: npm install
4. Run: npm run setup
5. Run: npm run echo-claude

Option B: Manual
----------------------------------
1. Extract this package to a folder
2. Open terminal in that folder
3. Run: npm install
4. Copy env.template to .env
5. Edit .env with your API keys
6. Run: npm run echo-claude

=====================================
QUICK COMMANDS
=====================================

npm install              Install dependencies
npm run setup           Run setup wizard
npm run echo-claude     Start Echo bot

pm2 start echo-claude-bot.js --name echo   Run 24/7
pm2 logs echo                              View logs
pm2 restart echo                           Restart
pm2 stop echo                              Stop

=====================================
PRODUCTION DEPLOYMENT
=====================================

For 24/7 operation:

1. Install PM2: npm install -g pm2
2. Start Echo: pm2 start echo-claude-bot.js --name echo
3. Save config: pm2 save
4. Auto-start: pm2 startup (follow instructions)

PM2 will:
✅ Auto-restart on crashes
✅ Keep Echo running on reboot
✅ Provide log management
✅ Enable remote monitoring
